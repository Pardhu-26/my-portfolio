# Pardha Sai – Personal Portfolio

A clean, responsive, professional portfolio website built with pure HTML, CSS, and vanilla JS.

## Folder Structure

```
portfolio/
  index.html    — Main HTML
  style.css     — All styles (dark theme, responsive)
  script.js     — Scroll reveal, typed text, animations
  resume.pdf    — Your resume (replace with actual file)
  README.md     — This file
```

## Setup

1. Replace `resume.pdf` with your actual resume file.
2. Update the LinkedIn URL in the Contact section of `index.html`.
3. Update GitHub links on project cards once repos are live.
4. Open `index.html` in any browser — no build step needed!

## Deploy on GitHub Pages

1. Push this folder to a GitHub repository (e.g. `pardhu-26.github.io`)
2. Go to repo → Settings → Pages
3. Source: Deploy from branch → main → / (root)
4. Save. Your site will be live at `https://pardhu-26.github.io`

## Deploy on Netlify

### Option A – Drag & Drop (fastest)
1. Go to https://netlify.com → Log in
2. Drag the entire `portfolio/` folder onto the Netlify dashboard
3. Done — live in seconds!

### Option B – Git integration
1. Push repo to GitHub
2. Netlify → New site from Git → choose your repo
3. Build command: (leave empty)
4. Publish directory: `.` (or the folder name)
5. Deploy
